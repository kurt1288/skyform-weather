<h1 align="center">Skyform Weather</h1>

<h4>Skyform is a free weather app aimed at providing weather in a more data-focused format rather than fancy-looking graphics.</h4>

<h2>Installation</h2>
<p>Skyform is a progressive web applicaton. It will work simply by visiting the <a href="https://kurt1288.github.io/skyform-weather/">website</a> in your phone's browser but you have the option to install it from there.</p>
<h3>On Android</h3>
Once on the site, tap the three-dot menu in the upper right corner, select "Add to Home screen", then "Install".
<h3>On iPhone</h3>
Tap the Share icon (its location may vary depending on the browser. In Safari it is on the bottom of the screen). Find and tap on "Add to Home Screen". Confirm the name and hit "Add".
